// Project 01 ML / NLP

Mental Health Text Classifier

Built and deployed a machine-learning NLP classifier to categorise Reddit posts into mental-health stress categories — DV, Financial, Loneliness, Relationship, and Work Stress. Implemented text preprocessing (stopword removal, lemmatisation), TF-IDF + n-grams + Bag-of-Words for feature engineering, and an XGBoost classifier — achieving 75% accuracy. Deployed via Flask on Render for real-time predictions. Tackled class imbalance using cross-validation and hyperparameter tuning.
